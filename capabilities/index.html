What I’m good at.

Android Engineering

I build modern Android apps that are reliable, maintainable, and pleasant to use. I’m comfortable with:

  • Jetpack Compose for UI
  • Dependency injection with Hilt
  • Multi-module app structures
  • Handling sensors and background behavior
  • Designing features that are both technically sound and easy to understand

SenseMap and Egg Timer are good examples of how I combine these pieces.

Developer Tools & System Architecture

I enjoy building tools that help developers understand and work with their code more easily. With SenseDev, I explored:

  • Turning code into a graph of relationships.
  • Visualizing project structure.
  • Using AI to assist, without replacing the developer.

This reflects how I my thought process functions in relation to software, as systems that benefit from better visibility and tooling.

AI-Augmented Interfaces

I am passionate about designing experiences and tools around AI. I care about:

  • Grounding answers in real data or code.
  • Making AI behavior predictable and explainable.
  • Using AI to highlight patterns or insights, not to overwhelm users.

SenseMap and SenseDev both use AI in ways that support understanding.

Product Thinking

Across all my projects, I try to think in terms of:

  • What problem this actually solves.
  • How clearly the user can understand what’s going on.
  • How the system will behave over time, not just in a demo.
  • How easy it will be to extend and maintain.
