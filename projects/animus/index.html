Animus

Talk to the world around you.

[LIVE DEMO →] [VIEW SOURCE →]

What Animus Is

AI-powered web app that identifies any physical object via camera, gives it a personality, and lets you have a full voice conversation with it as itself. Point at your microphone — it speaks its opening line out loud in its own voice. Speak back in any language, it responds in kind. Built in a weekend, deployed and live.

The Idea

What if the objects around you could speak? Not as assistants, not as tools — as themselves, with personality and perspective. It's a small experiment in what happens when AI understands the physical world well enough to give it a voice. The idea connects directly to larger work on spatial AI and augmented object intelligence.

What I Built

  • React + Vite frontend with a custom cyberpunk AR aesthetic — scanlines, glitch text, neon cyan/magenta, JetBrains Mono
  • Gemini Vision API for object identification and personality generation
  • Gemini Chat API for stateless conversation — history and detected language maintained client-side, passed per request
  • Groq Whisper (whisper-large-v3-turbo) for speech-to-text with automatic language detection — hold to speak, release to send
  • Groq Orpheus TTS (canopylabs/orpheus-v1-english) for natural, expressive voice synthesis — the object speaks its opening line automatically on link initiation
  • Multilingual voice pipeline — speak in any language, Whisper detects it, Gemini responds in the same language, TTS speaks back
  • Vercel serverless API routes: /api/gemini, /api/transcribe, /api/speak
  • Web Speech API fallback for non-English TTS when Orpheus (English-only) is not applicable

Architecture

Camera feed (browser MediaDevices API)
    ↓ tap / scan button
Frame capture → base64 JPEG
    ↓
/api/gemini — action: 'vision'
    Gemini identifies object, generates personality + opening line
    ↓
ObjectCard → [INITIATE LINK]
    ↓
ChatPanel opens — object speaks opening line via Orpheus TTS automatically
    ↓
Voice input: hold mic → MediaRecorder → /api/transcribe (Groq Whisper)
    returns transcript + detected language
    ↓
/api/gemini — action: 'chat'
    Gemini responds in character, in detected language
    ↓
/api/speak (Groq Orpheus) → object speaks back
    fallback: Web Speech API for non-English responses

Why It Matters

Animus is a proof of concept for what becomes possible when AI understands the physical world — not just answering questions about it, but reasoning about what's in view, generating contextual knowledge, and expressing it through voice. The multilingual voice pipeline demonstrates end-to-end audio AI: STT with language detection, language-aware LLM response, and neural TTS — the same stack that powers next-generation voice-first interfaces on wearables and smart glasses.

It's also the web prototype of ideas that now live in the ARCore depth + Gemini research project.
