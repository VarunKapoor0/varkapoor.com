AABB Visualization & Spatial Debugging

Computing and visualizing axis-aligned bounding boxes for engine-level spatial reasoning.

What it is

A PrimeEngine extension that computes and visualizes Axis-Aligned Bounding Boxes (AABBs) to support spatial debugging and engine-level reasoning about object bounds.

Why I built it

Bounding volumes are foundational for collision systems and rendering optimizations. I wanted a practical implementation that makes spatial math visible and verifiable in-engine.

What I built

  • Implemented AABB computation for objects in the scene.
  • Added a visualization layer so bounds can be inspected during runtime.
  • Used the visualization as a debugging tool for correctness and future system extensions.
